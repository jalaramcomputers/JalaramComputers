How to bulk upload products

  1. Download the Excel template below — it has the right columns and an example row.
  2. Fill in one product per row. name and price are required; everything else is optional.
  3. Save the file as .xlsx and upload it here.
  4. Slugs are auto-generated from the product name — you don't need to set them.
